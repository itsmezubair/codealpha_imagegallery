# CodeAlpha_ImageGallery - Advanced Responsive Image Gallery

An internship-level professional Image Gallery web application built with HTML5, CSS3, and Vanilla JavaScript. This project features a modern dark-themed glassmorphism UI with full responsiveness and advanced interactive features.

## 🚀 Features

- **Responsive Image Grid:** A clean and modern layout that adapts to all screen sizes (Desktop, Laptop, Tablet, Mobile).
- **Category Filtering:** Filter images by Nature, Cars, Technology, and Animals without page reloads.
- **Search Functionality:** Real-time search to find images by title or category.
- **Lightbox Preview:** Fullscreen image preview with smooth fade and scale animations.
- **Next/Previous Navigation:** Navigate through the gallery items using on-screen buttons or keyboard arrow keys.
- **Dark/Light Mode:** Toggle between a premium dark theme and a clean light theme (persisted via localStorage).
- **Smooth Animations:** Hover effects, scroll reveals, and glassmorphism UI elements.
- **Glassmorphism UI:** Modern backdrop filters and translucent cards for a premium look.
- **Performance Optimized:** Lazy loading for images and lightweight code.

## 🛠️ Technologies Used

- **HTML5:** Semantic structure.
- **CSS3:** Custom properties (variables), Flexbox, CSS Grid, Media Queries, and Keyframe animations.
- **Vanilla JavaScript:** ES6+, Intersection Observer API, and DOM manipulation.
- **Google Fonts:** Poppins.
- **Font Awesome:** For professional icons.
